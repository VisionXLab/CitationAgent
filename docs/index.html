Overview · 概览

What is CitationAgent?

CitationAgent 是一款面向科研人员的论文被引画像分析智能体。只需输入论文题目（或从 Google Scholar 学者主页批量导入），系统将自动完成：爬取所有施引文献、识别著名学者背景（院士、Fellow、杰青等）、搜索引用原句，最终生成一份包含关键词云、引用趋势、国家分布、学者画像等多维可视化内容的精美 HTML 报告，无需服务器，单文件即可分享。

CitationAgent is a paper citation portrait analysis agent for researchers. Given a paper title or Google Scholar profile, it automatically crawls all citing papers, identifies renowned scholars (academicians, Fellows, etc.), retrieves citation sentences, and produces a fully self-contained HTML portrait report.
Features · 核心功能

Key Capabilities

从爬取到报告，全流程自动化

🎓
著名学者自动识别
自动识别中国科学院/工程院院士、IEEE/ACM/ACL Fellow、国家杰青、长江学者等重量级学者，颜色标注，一目了然。
📊
多维可视化报告
关键词云、引用趋势预测、国家/机构分布、学者画像、引用位置与情感分析，单 HTML 文件，无需服务器。
🔍
引用原句精准检索
对施引文献逐篇搜索引用原句，定位所在章节（引言/方法/实验），支持 Markdown 渲染展示。
🔁
断点续爬 · 缓存复用
任务中断后可从断点继续；已搜索过的学者信息自动缓存，多次分析无需重复调用 LLM，大幅降低费用。
📈
突破千篇限制
开启年份遍历模式后，按年份分段爬取并合并去重，高被引论文（如 Transformer、BERT）同样可获取完整数据。
🤖
LLM 综合 Insight
自动生成四维结构化总结——引用规模与分布、主要用途、代表性原文摘录、综合说明，支持任意 OpenAI 兼容 API。
Pipeline · 使用流程

How It Works

四步完成完整的被引画像分析

1
配置 API
填入 ScraperAPI Key 与 OpenAI 兼容 API Key，支持 V-API、OpenAI 等任意服务。
2
输入论文
直接粘贴论文题目，或粘贴 Google Scholar 主页 URL 批量导入。
3
选择层级
从全面版到标准版，按需平衡分析深度与 token 消耗，点击开始分析。
4
获取报告
下载 paper_dashboard.html，浏览器直接打开即可查看完整多维画像报告，可直接分享。
